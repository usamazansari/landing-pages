A collection of landing pages for many websites using Mantine and Astro


## Installation

- Install [Bun](https://bun.sh/)
  ```
  curl -fsSL https://bun.sh/install | bash
  ```
- Install dependencies
  ```
  bun install
  ```
- Start the local DEV server
  ```
  bun start
  ```
